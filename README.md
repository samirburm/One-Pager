# HYBRID.

## Projektbeschreibung

HYBRID. ist eine moderne One-Page-Website zum Thema Hybrid Training. Ziel des Projekts war die Entwicklung einer ansprechenden und responsiven Website mit HTML, CSS und JavaScript. Die Website informiert über die Trainingsphilosophie eines Hybrid Athletes und stellt die Bereiche Krafttraining, Ausdauertraining, Mobility und Recovery vor.

Besonderen Wert habe ich auf ein modernes Design gelegt. Die Website verwendet ein dunkles Farbschema, grossflächige Bilder, Kartenlayouts und Animationen. Zusätzlich wurde mit JavaScript ein Light-/Dark-Mode-Schalter sowie eine interaktive Join-Section umgesetzt. Die Website passt sich verschiedenen Bildschirmgrössen an und bietet eine benutzerfreundliche Navigation.


## Was hat beim Projekt gut funktioniert?

Besonders gut funktioniert hat das Erstellen des Grundgerüsts mit HTML sowie die anschliessende Gestaltung mit CSS. Durch die klare Struktur konnte ich die einzelnen Bereiche der Website schrittweise aufbauen und erweitern.

Besonders zufrieden bin ich mit dem dunklen Design der Website, da es gut zum Thema Hybrid Training passt und einen modernen Eindruck vermittelt. Auch die Kombination aus Bildern, Cards und Animationen trägt dazu bei, dass die Website ansprechend und professionell wirkt.


## Wo bist du an Grenzen gestossen?

Bereits zu Beginn des Projekts bin ich bei der Einrichtung von Git und GitHub an Grenzen gestossen. Das Verknüpfen des lokalen Projekts mit GitHub war für mich neu und erforderte einige Versuche, bis alles korrekt funktionierte.

Auch bei der Umsetzung der CSS-Animationen musste ich mich zuerst einarbeiten. Dabei war es anfangs nicht immer einfach zu verstehen, welche Eigenschaften für die gewünschten Effekte verantwortlich sind und wie die einzelnen Animationen aufgebaut werden.

Die grösste Herausforderung war jedoch JavaScript. Da ich bisher nur wenig Erfahrung damit hatte, musste ich zunächst verstehen, wie JavaScript mit HTML und CSS zusammenarbeitet. Besonders beim Light-/Dark-Mode und bei den interaktiven Elementen war es wichtig nachzuvollziehen, welche Funktionen was bewirken und wie auf einzelne Elemente der Website zugegriffen werden kann.

Zusätzlich stellte die responsive Gestaltung der Website eine Herausforderung dar. Mithilfe von Media Queries musste sichergestellt werden, dass die Website auf verschiedenen Bildschirmgrössen korrekt dargestellt wird und die Navigation, Bilder und Inhalte auch auf mobilen Geräten übersichtlich bleiben.


## Was hast du dabei gelernt, das du nicht erwartet hättest?

Während des Projekts hat mich überrascht, wie viel man bereits mit reinem CSS umsetzen kann. Besonders die Animationen wirken zwar auf den ersten Blick einfach, machen aber bei der Nutzung der Website einen grossen Unterschied und tragen dazu bei, dass die Website moderner und professioneller wirkt.

Ausserdem hat mich überrascht, wie viele Gestaltungsmöglichkeiten CSS bietet. Durch den Einsatz von Cards, Grid-Layouts und Hover-Effekten konnte ich ein modernes Design erstellen.

Ebenfalls nicht erwartet hatte ich, wie sicher ich im Verlauf des Projekts im Umgang mit HTML und CSS geworden bin. Auch JavaScript war für mich zu Beginn neu. Durch die Umsetzung des Light-/Dark-Modes und der interaktiven Elemente habe ich jedoch ein besseres Verständnis dafür entwickelt und verstehe nun die Grundlagen von JavaScript deutlich besser.
